# Dropdown UI

## Usage

```html
<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8" />
    <title>Dropdown</title>
    <link rel="stylesheet" type="text/css" href="dropdown.min.css" />
</head>
<body>
    <div id="drdw"></div>
    <script type="text/javascript" src="dropdown.js"></script>
    <script>
        const data = [{...}];
        const dropdown = new Dropdown("#drdw", { 
            items: data,
            // options
        });
    </script>
</body>
</html>
```
